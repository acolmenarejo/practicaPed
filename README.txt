# practicaPed

SISTEMA OPERATIVO: LINUX, Opensuse
HERRAMIENTAS PROGRAMACION: Pycharm, vi, terminal opensuse
LENGUAJE: Python 3.8

PROCEDIMIENTO:

1. Ejecutar por consola en la ubicacion del proyecto: python3 _main.py
2. Seleccionar 1 para ejecutar el cliente
3. Seleccionar 2 para ejecutar el servidor
4. Ejecutar por consola: python3 tdd_final.py para ejecutar los tests
